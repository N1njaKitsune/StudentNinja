IMPORTANT — WHAT THIS FLOW IS
Everything on the live site currently lives inside Mushin (the breathing discipline). The "Hub" is an element selector (Kaze/Chi/Mizu/Hi), not a discipline selector. There is no screen where students choose between Metsuke, Mushin, and Tachi. The entire prototype is one discipline deep.
The Gap — What the Architecture Requires but Does Not Exist
Missing Layer: Nendō Home
The architecture calls for a Nendō Home screen — the hub where students see three doors: Metsuke (blue), Mushin (green), Tachi (red). This is the top-level navigation layer that sits between the Student Profile entry and each discipline's content. It does not exist on the live site. Without it, students cannot reach Metsuke or Tachi at all.

HTML prototypes exist: Nendo_Home_v1.0.html v2.0_Painted.html — but these show 3 disciplines, not the 4 elements currently on the live Hub.
Missing: Metsuke Hub
No Metsuke screen exists on the live site. HTML prototypes exist: MetsukeHub_v1.0.html v2.0.html — but nothing has been implemented.
Missing: Tachi Hub
No Tachi screen exists on the live site. HTML prototype exists: TachiHub_v1.0.html — earliest stage of the three disciplines.
